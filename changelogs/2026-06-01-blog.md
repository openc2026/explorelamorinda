Not doing it today — single-improvement budget already spent.
- **The test script's meta-description regex has a real bug** that costs us on any blog post whose frontmatter description starts with an apostrophe-containing word (e.g. "It's"). The regex `[^"']*` truncates at the first single quote inside a double-quoted attribute. Caught it on this post and rewrote the description to avoid; the fix on the test side is to replace the character class with `[^"]*` for the double-quoted case (since the inner content can legitimately contain apostrophes). Worth filing as a follow-up; not fixing in this PR because (a) it's out of scope for the blog daily and (b) any test-script change should be its own reviewable PR.
- **The "Seasonal Note" pattern on things-to-do/_index.md needs a weekly maintenance cadence going forward.** It's the most visibly time-stamped block on the hub and goes stale fast — the prior block referenced "this weekend" for an event that was a full week in the past. Suggest making the weekly seasonal refresh a standing improvement target for any blog-daily run where the hub block is more than 5-7 days old. Today's edit applies the durable-framing lesson from yesterday's Loard's / Chef Chao work where applicable (e.g. "First Week of June" rather than "Memorial Day Weekend → Early June"), while keeping the dated specifics that actually have to be dated (June 5 Acalanes finish, June 8 camp open, June 11 concert opener, June 21 Father's Day, June 20 solstice).
- **Cross-link mesh strengthened.** New blog post anchors into Lafayette Reservoir, Tunnel Psychology, Caldecott Tunnel guide, and yesterday's Sunday post. Things-to-do seasonal block adds an outbound link to the new June Gloom post and the existing Farmers Market Theater post. Net new edges: 2 from hub to blog, 3 from new blog to existing pages.
